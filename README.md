# TrafficLightAlgorithm

## 内容
C#言語で信号機のアルゴリズムを展開したプログラムを作成する  
C#言語を意識したコーディングの作成  
Windowsフォーム（.NET Framework）で作成し視覚的にも信号を
イメージした作成にする  
作成は、オブジェクト指向を意識した内容で行うこと  

## 目的・狙い
Excel VBAで作成した同様の動作をC#言語で作成することで  
プログラミングの効率化、オブジェクト指向、非同期処理 の理解を
深めて貰い自身の力だけでプログラミング出来るようにスキルアップをする  
  
また、VBAで作成したコードをC#へ書き直すことによりVBAとC#の違い、
C#言語の理解が深まることが期待出来ます。

## 課題
以下仕様の交差点の交通信号機（現存するもの同様）を表現するプログラムを作成してください。  
- 交差点は縦方向（南北）と、横方向（東西）の4方向からの車道が行き交うものです。  
- 信号機は、それぞれの車道にひとつずつ設置されており、点灯種類は、「赤」「黄」「青」の3種類です。  
- 信号機は、縦方向の進行と、横方向の進行が、同時にならないように制御し、進行可能時間、及び
停止時間をシステム使用者が簡単に変更できるようにしてください。
また、進行可能方向の切り替わりタイミングで、全車両を安全に停止させるために、
すべての方向の信号機が「赤」になる準備時間を設けてください。
- 作成は、最終的にクラスを意識したプログラムにしてください。
- 車用信号完成、「歩行者用信号」の動きも追加してください。（青点滅後、赤に切り変わる）
